# 5-Email Lifecycle Sequence

## Audience

New download/trial users who have not yet formed a stable workflow in Skilly.

## Email 0 (Immediately): Welcome + 3-Minute Setup

- Subject: Your Skilly setup starts now (3 minutes)
- Goal: first launch + first skill install
- Primary CTA: Install your first skill

Body points:
- What Skilly is and who it is for
- 3-step quick-start
- Direct link to first action

## Email 1 (Day 1): First Quick Win

- Subject: Your first workflow win with Skilly
- Goal: validate immediate value
- Primary CTA: Apply a preset

Body points:
- One practical workflow example
- Before/after time comparison
- One action to reproduce

## Email 2 (Day 3): Common Failures and Fixes

- Subject: Avoid these 3 setup mistakes
- Goal: reduce early frustration
- Primary CTA: Run a quick skill audit

Body points:
- Top three mistakes
- One fix for each mistake
- Link to template page

## Email 3 (Day 6): Multi-Repo Workflow Template

- Subject: A repeatable multi-repo setup template
- Goal: promote retained usage
- Primary CTA: Enable global + project split

Body points:
- Global skills vs project overrides
- Weekly maintenance checklist
- One template users can copy

## Email 4 (Day 10): Upgrade Value and Long-Term Gain

- Subject: Keep your workflow consistent as you scale
- Goal: strengthen upgrade intent and long-term retention
- Primary CTA: Activate your lifetime license

Body points:
- Long-term maintenance benefits
- Typical switching objections + responses
- Direct upgrade/activation CTA

## Rules

- One email, one primary CTA
- Clear > clever subject lines
- No fluff; concrete next action in every email
