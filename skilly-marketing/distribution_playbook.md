# Distribution Playbook (Zero Budget)

## Channel Priority

1. Website SEO content
2. X short-form + threads
3. Reddit value responses
4. GitHub release notes/discussions
5. Dev.to mirrors
6. Milestone launches: Hacker News, Product Hunt

## Repurposing Rule (per long-form article)

Create and publish all 4 derivatives within 72 hours:

1. Tutorial post
2. Mistake/pitfall post
3. Comparison post
4. Checklist post

All derivatives should point to the same target URL with unique `utm_content`.

## X Posting Framework

- Hook: pain or contradiction
- Body: failed old way -> practical method
- Proof: concrete workflow step
- CTA: free trial/download

## Reddit Posting Framework

- Start with context from first-hand experience
- Give complete value in the post itself
- Link only at the end as optional reference
- Prioritize comments on existing threads over self-posting

## GitHub Distribution Framework

- Weekly release note with one workflow takeaway
- Discussions post for feedback request
- Keep language practical and implementation-first

## Dev.to Mirror Rules

- Reuse canonical blog structure
- Add rel-canonical to Skilly site URL where possible
- Keep title aligned with target keyword

## Drop Rules

Drop a template/channel if either condition is true for 3 consecutive weeks:

- CTR < 1%
- No measurable download clicks
