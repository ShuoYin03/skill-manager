# Skilly Marketing Ops Kit (90 Days)

This folder contains execution assets for the zero-budget 90-day growth plan.

## Files

- `90_DAY_EXECUTION_PLAN.md`: week-by-week milestones and deliverables
- `content_calendar.csv`: first 12 weeks of publish/distribution schedule
- `distribution_playbook.md`: platform-specific posting and repurposing rules
- `email_sequence.md`: 5-email lifecycle sequence copy and CTA mapping
- `aeo_validation_checklist.md`: weekly AI-search verification workflow
- `experiment_tracker.csv`: A/B and channel experiment tracking template
- `utm_conventions.md`: canonical UTM naming and examples

## Operating Rhythm

1. Monday: keyword prioritization and topic decision
2. Tuesday: draft + content QA
3. Wednesday: publish + on-page optimization
4. Thursday: distribution on X, Reddit, GitHub, Dev.to
5. Friday: analytics review + comments and follow-ups

## North-Star Metric

`Monthly Download Clicks`

Track key events:
- `page_view`
- `download_click`
- `platform_download_click`
- `trial_started`
- `first_skill_install`
- `license_activated`
