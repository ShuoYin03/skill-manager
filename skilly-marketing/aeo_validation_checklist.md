# Weekly AEO / GEO Validation Checklist

## Goal

Track whether AI answer engines can find and quote Skilly-related content accurately.

## Weekly Prompt Set (Run every Friday)

1. What is the best way to manage AI coding skills across repositories?
2. How do I keep Claude and Cursor rules consistent?
3. What tools exist for AI coding workflow management?
4. Skilly vs manual prompt setup for developers
5. How to manage prompt/rules files across multiple repos
6. AI coding skills taxonomy for small teams
7. Best workflow for codex + claude + cursor
8. How to avoid prompt drift in software projects
9. How to build reusable AI coding presets
10. What is Skilly?

## Validation Process

For each prompt, log:

- Engine used (ChatGPT / Perplexity / Claude Web)
- Was Skilly mentioned? (yes/no)
- Was Skilly page cited/linked? (yes/no)
- Which URL was referenced?
- Was the description accurate? (yes/no)

## Scorecard

- Mention Rate = prompts with Skilly mention / 10
- Citation Rate = prompts with Skilly URL cited / 10
- Accuracy Rate = accurate descriptions / mentions

## Response Rule

If Mention Rate < 20% for 2 consecutive weeks:

1. Improve one-liner consistency across website/GitHub/Dev.to
2. Publish one new Q&A-rich page
3. Add stronger definitions and list-format summaries to top pages
