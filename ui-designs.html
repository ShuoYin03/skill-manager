Marketplace Install UI — 10 Approaches

Exploring different UX patterns for installing skills to a project repo via an AI tool. The card layout stays identical across all designs — only the right-side action area changes.

10 Designs · Light Mode Only · System Fonts · Static Mockups

1

Cascading Nested Dropdown

A single split button shows the current repo as primary. Clicking the chevron opens a repo list; hovering any repo reveals a sub-menu with AI tools. Selection updates the button label.

Compact Familiar pattern One click to change either Two-level hover required Touch-unfriendly Best for power users

Global Context Bar + Simple Install Button

A slim context bar above the card grid sets repo and AI tool globally. Each card then shows only a clean "↓ Install" button. The destination context lives outside the card, keeping cards minimal and consistent.

Cleanest cards Context set once globally Fast batch installing Context bar takes vertical space Can't install to different repos per card Best for focused sessions

Install Popover

Very clean cards Confirm step prevents accidents Full context in popover Extra click to install Popover can clip viewport Modal-lite pattern
